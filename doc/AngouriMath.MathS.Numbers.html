Class MathS.Numbers

All operations for and its derived classes are available from here.

These methods represent the only possible way to explicitly create numeric instances. It will automatically downcast the result for you, so 

Number.Create(1.0);
is an . To avoid it, you may temporarily disable it

 MathS.Settings.DowncastingEnabled.As(false, () =>
{
   var yourNum = Number.Create(1.0);
});
